=head0 Функции

X<function>
X<subroutine>

I<Функция> (или I<подпрограмма>) в Perl - это обособленная, инкапсулированная
часть исполнения. Программа - это набор маленьких чёрных ящичков, где
взаимодействие этих функций управляет ходом выполнения программы. Функция может
иметь имя. Она может потреблять входные данные. Она может выдавать результат на
выходе.

Функции - это главный механизм абстракции, инкапсуляции и повторного
использования в Perl 5.
